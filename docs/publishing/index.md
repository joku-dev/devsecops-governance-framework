# Publishing Documents

## Purpose

This folder contains publication-oriented documents that can be reused outside
the repository documentation, for example in Confluence, management portals or
internal knowledge bases.

Publishing documents are not source documents unless they are explicitly moved
through the source-document intake process. They are communication artifacts
derived from, or aligned with, the governed repository content.

## Documents

| Document | Purpose |
|---|---|
| `docs/publishing/confluence-governance-repo-artikel.md` | German Confluence article explaining the purpose, value and operating model of the DevSecOps governance repository. |

## Usage Guidance

Use publishing documents when stakeholders need a narrative explanation of the
repository outside the MkDocs site.

If a publishing document starts to define policy, control requirements,
architecture rules or baseline behavior, it must be reviewed as a governance
source candidate before being treated as authoritative.
