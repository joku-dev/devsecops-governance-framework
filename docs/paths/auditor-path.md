# Auditor Path

## Who This Is For

Use this path if you need to assess governance intent, traceability, verification logic, or evidence quality.

## Goal

After following this path, you should understand:

- where requirements come from
- how requirements map into controls
- how controls are verified
- what the current operational state shows

- which document authorized this control
